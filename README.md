# 🧮 Desafio POO em PHP - ParcelasJá

Este projeto foi desenvolvido como parte de um desafio para explorar habilidades de **Programação Orientada a Objetos (POO)** em **PHP 8.2+**.

O objetivo é implementar três classes principais:
- `Shape` (classe base)
- `Rectangle` (herda de Shape)
- `Circle` (herda de Shape)

Cada classe demonstra o uso de **herança**, **encapsulamento**, **constantes**, **construtores** e **métodos personalizados**.

---

## 🚀 Requisitos

- **PHP 8.2+**
- **Composer**

---

## 📦 Instalação e Configuração

1. Clone o repositório ou faça o download:
    ```bash
    git clone https://github.com/seu-usuario/Desafio_ParcelaJa_POO-PHP.git
    cd Desafio_ParcelaJa_POO-PHP
    ```

2. Instale as dependências e gere o autoload:
    ```bash
    composer install
    composer dump-autoload
    ```

3. Certifique-se de que a estrutura de pastas está assim:
    <pre markdown="1">
    Desafio_ParcelaJa_POO-PHP/
    ├── Classes/
    │   ├── Shape.php
    │   ├── Rectangle.php
    │   └── Circle.php
    ├── index.php
    ├── composer.json
    └── vendor/
    </pre>
